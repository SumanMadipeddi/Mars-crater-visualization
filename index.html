Mars Crater Visualization

Description:

This visualization maps the geographical distribution and attributes of craters on Mars. It employs a heatmap to depict crater density, visualized through a gradient color scale, with warmer colors indicating higher densities. Each crater is represented by a circle, and the depth is represented by a dynamic "lollipop" marker that extends vertically from the crater's base point on the map to demonstrate depth. The head of the lollipop changes color based on the stratigraphic layers of the crater. Interactive elements include tooltips, which reveal detailed information about each crater upon mouse hover, such as diameter, depth, and stratigraphic composition.

Custom Visualization Category: Combinatorial

This visualization is categorized as "Combinatorial" because it blends several established D3 techniques and visual metaphors. It utilizes heatmaps to show spatial density, circles to represent crater diameters, and colors to indicate different layers. Additionally, 'lollipop' markers provide depth exploration, complemented by interactive tooltips that offer detailed data insights. This integration creates a multi-layered and interactive experience that deepens user understanding and transcends conventional static visualizations.

Modifications:

This visualization modifications include:

  • Integration of a novel "lollipop" marker design to encode crater depth and stratigraphic data dynamically.
  • Enhancement of the heatmap to serve as a base layer, illustrating crater density with a customized color gradient.
  • Addition of interactive tooltips that provide detailed data specifics dynamically upon interaction.
  • Custom legend design that explains both the heatmap's color gradient and the stratigraphic layering of craters.
